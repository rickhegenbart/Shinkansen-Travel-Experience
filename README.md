# Shinkansen-Travel-Experience
End-to-end machine learning project predicting Shinkansen passenger satisfaction using travel and survey data. Includes data analysis, feature engineering, model comparison, CatBoost tuning, cross-validation, feature importance, and test predictions, achieving 95.7% validation accuracy.
